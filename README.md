# TriageDeck Releases

This repo is just for downloadable TriageDeck builds.

TriageDeck is a portable Windows diagnostics toolkit I made because I got tired of needing a pile of different tools just to figure out what a PC was trying to say.

The main goal is not just showing raw data. It is helping people actually understand what that data means.

Here you will find:

- release zip files
- version notes
- simple download updates

If TriageDeck saves you time or helps you rescue a PC, please consider supporting the project!:
