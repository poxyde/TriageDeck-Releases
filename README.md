# EventLens Releases

This repo is just for downloadable EventLens builds.

EventLens is a portable Windows diagnostics toolkit I made because I got tired of needing a pile of different tools just to figure out what a PC was trying to say.

The main goal is not just showing raw data. It is helping people actually understand what that data means.

Here you will find:

- release zip files
- version notes
- simple download updates

If EventLens saves you time or helps you rescue a PC, please consider supporting the project!:
